# ⏰ Alarm Clock Web App

A clean, responsive, and easy-to-use **Alarm Clock App** built by **SAROJ DHITAL** using **HTML, CSS, and JavaScript**.  
This project provides a smooth way to set alarms, manage them, and stay on top of your schedule with a simple, modern interface.
***
## 🛠 Technologies Used

- 🖥 **Frontend:** HTML, CSS (soft gradients, clean layout, readable typography)  
- ⚙️ **Logic:** JavaScript (timers, alarm scheduling, audio handling)
***
## 🔑 Features

- ✅ Real-time digital clock display  
- ✅ Add multiple alarms  
- ✅ Alarm rings with sound  
- ✅ Delete alarms anytime  
- ✅ Toggle alarm ON/OFF  
- ✅ Prevents duplicate alarms  
- ✅ Smooth UI interactions  
- ✅ Fully responsive design  
***
## ▶️ How It Works

1. The digital clock updates every second.  
2. You select hour, minute, and AM/PM to create a new alarm.  
3. Alarms get added to a list and activate at the exact time.  
4. When an alarm matches the current time, the ringtone plays.  
5. You can stop the alarm, toggle it, or delete it from the list.
***
